# Support

AudioManager is an early pre-release project. Support is best-effort and no
service-level commitment is offered.

- Use GitHub Issues for reproducible, non-security defects and feature ideas.
- Use [SECURITY.md](SECURITY.md) for suspected vulnerabilities; never disclose
  sensitive details in a public issue.
- Consult the setup, streaming, and troubleshooting guides before reporting an
  audio-routing problem.

Do not attach private audio, credentials, or device identifiers to public
reports.
